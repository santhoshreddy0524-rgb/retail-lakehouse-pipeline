# Retail Lakehouse Pipeline

This project demonstrates an end-to-end data engineering pipeline using Databricks, PySpark, Delta Lake, and SQL.

## Concepts Covered

- Batch and incremental processing
- Full load
- CDC batch apply
- Bronze, Silver, Gold architecture
- Audit columns
- Control tables
- Run log tables
- Watermarking
- Lookback window
- MERGE/UPSERT
- Deduplication
- Data quality checks
- Quarantine tables
- File arrival checks
- Orchestration and scheduling
- Retry logic
- Idempotency
- Backfill and reprocessing
- Partitioning and partition pruning
- Schema evolution and enforcement
- Performance optimization
- Monitoring and alerting
- SLA and freshness
- Security and governance
- Data lineage
- Metadata-driven pipelines
